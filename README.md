# Diplom_1 проект: Юнит-тестирование

Проект предназначен для автоматизированного юнит-тестирования основных компонентов веб-приложения [Stellar Burgers](https://stellarburgers.nomoreparties.site/).

## Технологии

| Технология            | Версия       | Назначение                                                                |
|-----------------------|--------------|---------------------------------------------------------------------------|
| Java                  | 11           | Язык программирования, используемый для написания тестов                  |
| Maven                 | 3.9.0        | Система управления зависимостями и сборкой проекта                        |
| JUnit                 | 4.13.2       | Фреймворк для написания и выполнения модульных тестов                     |
| Mockito               | 1.10.19      | Библиотека для создания моков в тестах                                    |
| JaCoCo Maven Plugin   | 0.8.7        | Инструмент для анализа покрытия кода тестами                              |
